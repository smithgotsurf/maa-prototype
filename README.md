# MAA Prototype

A registration and information web app prototype for the **Meadow Athletic Association (MAA)**.

Built with React + Vite. All app code lives in `src/App.jsx`.

## Sports Offered

### Spring
- T-Ball (Coed, ages 3–4)
- T-Shirt (Coed, ages 5–6)
- Baseball — 8U, 10U, 12U (boys)
- Softball — 8U, 10U, 12U (girls)

### Fall
- Soccer — 6U Coed, 8U Boys, 8U Girls
- Baseball — 8U, 10U, 12U (boys)
- Softball — 8U, 10U, 12U (girls)

### Winter
- Basketball — 6U Coed; Boys 8U/10U/12U/15U; Girls 8U/10U/12U
- Volleyball — 8U, 10U, 12U

## Registration Windows

| Season | Opens | Closes |
|---|---|---|
| Spring | Mid-January | End of February |
| Fall | Early July | Early August |
| Winter | Early October | Early November |

- Late registration adds **$20** if space is available
- Age groups may close early once full

---

## Season Schedules

### Spring
- Practices begin in **March**
- Games run **April – late May/early June**
- Game days: **Monday, Tuesday, Thursday**
- T-Ball games: **6:30 PM** · T-Shirt games: **7:15–7:30 PM**

### Fall
- Practices begin **late August**
- Games start **mid-September**
- Game days: **Monday, Tuesday, Thursday**

### Winter
- Practices begin **early December**
- Games start **early January**
- Volleyball games: mostly **Saturday**, some Tuesday/Thursday

## Field Rentals

Two fields available when not in use for games or practices.

| Field | Supported Divisions |
|---|---|
| **Field 1** | T-Ball, T-Shirt, 8U/10U/12U baseball & softball |
| **Field 2** | T-Ball, T-Shirt, 8U/10U baseball, 8U/10U/12U softball (no 12U baseball) |

**Pricing:**
- Without lights: **$15/hr**
- With lights: **$35/hr** (+$20/hr for lights)
